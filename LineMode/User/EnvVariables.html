Environment Variables

Environment variables are used for customizing the Line Mode Browser and the W3C Reference Library for a particular user's requirements. Under unix, environment variables are strings of the form: 

	name=value

They are used to pass information to programs to modify how they should run. The UNIX kernel never looks at these strings - their intepretation is up to the various applications.
