Line Mode Browser 4.0 Release Notes

Alpha Release 4, October 31 1995

  • Introduced "RELOAD" as command. If forces reload in memory and update of cache
  • Fixed bug where -single didn't have any history list
  • Changed the default help page to be online manual
  • Introduced POST as METHOD
  • Replaced the original event loop with the new version
  • Cleaned up and reorganized the code
  • Introduced CLEAR and FORWARD as commands
  • Introduced -delete as command line flag
  • Introduced -nt command line flag
  • Introduced -cl flag for counting content length

Henrik Frystyk, libwww@w3.org, October 1995
