WWW Line Mode Browser

VERSION 3.1 PRE 3 IS AVAILABLE NOW (July 13)! Release 3.1 will be out shortly. Comments are welcome at libwww@w3.org

The Line Mode Browser (www) is a C based character based World-Wide Web Browser. It is developed for use on dumb terminals and as a test tool for the W3C Reference Library. It can be run in interactive mode, non-interactive mode and as a proxy client. In non-interactive mode it can be used as a powerful tool in CGI scripts for the HTTPD. Furthermore it gives a variety of possibilities for data format conversion, filtering etc. It is primarily intended as a test-tool for quick access to the Web or used in batch jobs.

Getting the Source Code
The latest source code is available in the following formats:
  • compressed tar file
  • gzip'ed tar file
  • zip file
You can find old versions of the source code. All this and many other things you can find in our source archives and you can browse through the latest source files on our server!

A packet contains the following components:

  • The code definitions (*.c)
  • The interface definitions (*.h)
  • The documentation (*.html)
  • A Makefile for building the Line Mode Browser

You can see how to unpack and compile the software in the README file, and you can find more information on the arthitecture in the Internals and how to use it in the User's Guide.
