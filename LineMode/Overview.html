WWW Line Mode Browser

The Line Mode Browser (www) is a character based World-Wide Web Browser. It is developed for use on dumb terminals and as a test tool for the Common Code Library. It can be run in interactive mode, non-interactive mode and as a proxy client. In non-interactive mode it can be used as a powerful tool in CGI scripts for the HTTPD. Furthermore it gives a variety of possibilities for data format conversion, filtering etc. It is primarily intended as a test-tool for quick access to the Web or used in batch jobs.

Platforms
The browser is known to compile on the following Platforms. See also the VMS version.
Getting the Source Code
The LineMode Browser is available in two versions:
  • A singlethreaded version 2.15 as WWWLineMode.tar.Z
  • A multithreaded version 3.0 as WWWLineMode_3.0.tar.Z.
Both source packets can be found in the Source Archive.
Precompiles Binaries
Binaries are available for many Platforms as www*.tar.Z.
Copyright
