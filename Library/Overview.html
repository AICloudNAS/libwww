W3C Reference Library

The W3C Reference Library (libwww) is a general-purpose Web API written in C. It is a highly modular extensible API which can be used as the code base for writing Web clients, servers, proxies etc. Libwww provides reference code for HTTP and other Internet protocols and also for handling many media types including HTML.

If you are a W3C member then please jump to the our member area for the very latest version.

News and Updates - 4.1 beta 5 has been released August 25

Libwww 4.1 beta 5 is a minor update from version 4.1b4. The main difference is some bug fixes after intensive testing against the Common Lisp Server. The release can be downloaded now. As version 4.1b4, it includes a HTTP/1.1 conforming test implementation and many other new features.

Getting the Source Code
The latest source code is available in the following formats:
  • compressed tar file (ca. 1.8M)
  • gzip'ed tar file (ca. 1.2M)
  • zip file (ca. 1.5M)

You can find old versions of the source code and diffs together with many other things on our FTP server and you can browse through the latest source files on our server! A packet contains the following components:

  • Installation guide
  • Source code
  • Platform dependent make files
  • Hypertext documentation
  • Copyright statements and other legal mumbo jumbo

You can see how to unpack and compile the software in the README file, and you can find more information on how the Library is designed in the Library Architecture and how to use it in the User's Guide.
