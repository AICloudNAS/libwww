Libwww - the W3C Sample Code Library

Libwww is a general-purpose client side Web API written in C for Unix and Windows (Win32). With a highly extensible and layered API, it can accommodate many different types of client applications including clients, robots, etc. The purpose of libwww is to provide a highly optimized HTTP sample implementation as well as other Internet protocols and to serve as a testbed for protocol experiments.
