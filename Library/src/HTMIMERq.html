MIME Request Stream

The MIME Request stream generates a MIME request header and writes it to the target which is normally a HTWriter stream.

This module is implemented by HTMIMERq.c, and it is a part of the W3C Reference Library. 

#ifndef HTMIMERQ_H

#include "HTStream.h"
#include "HTReq.h"
