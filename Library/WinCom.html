Windows GUI PUT Example PUT Tool

The WinCom is a small MFC based example application that integrates libwww to perform HTTP/1.1 PUT of arbitrary file. I am not a GUI programmer and much can be improved - please try it out and feel free to improve and send back the new features! What you need to get going is
  • The latest libwww 5.1.l distribution with this minor patch to apply a few new functions
  • A new set of compiled def files that supports the new functions to replace the ones in "w3c-libwww-5.1l/Library/src/windows"
  • The source code of the WinCom itself to be installed in "w3c-libwww-5.1l/WinCom".

Have fun!

Nearby you can find the following example applications that you get when you download the libwww distribution:

  • The Mini Robot - a fast robot that allows you to check links etc.
  • The Line Mode Browser - a terminal based, simple browser
  • The Command Line Tool - a quick too to do all your Web work

Also have a look at the W3C Libwww activity statement on support, usability etc.
