This privilege is reserved for libwww collaborators who have intimate knowledge of the code base and have proven their ability to use this privilege to the benefit of the libwww user community.

How can I Sign Up?

The best way to so this is to start contributing patches based on the latest code base and to be active on the <www-lib@w3.org> mailing list. Make sure that you have read the libwww style guide before you start.
