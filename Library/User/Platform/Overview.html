Platforms Supported

This list gives an overview of the platforms where the W3C Reference Library, the HTTP Server, and the Line Mode Browser are known to compile. The Server and Line Mode Browser are both available in precompiled binaries for all the platforms marked binary=YES.

Thanks a lot to the big group of creative contributors for the long list of hints, patches and additions to the make files and system dependent code!

If your platform is not mentioned below then have a look at how to port the source code to your platform.
