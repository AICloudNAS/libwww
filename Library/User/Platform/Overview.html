Platforms Supported for libwww

This list gives an overview of the platforms where the W3C Reference Library is known to compile. Thanks a lot to the big group of creative contributors for the long list of hints, patches and additions to the make files and system dependent code! If you have any problems or questions then send a mail to libwww@w3.org or to the www-lib@w3.org mailing list. We also keep an archive of the mailing list so that more easily can get up to speed.
