Notes About HP

These are the installation and user notes for the W3C httpd, the W3C Reference Library and the Line Mode Browser. Any comments and suggestions are welcome at libwww@w3.org.

Problems with gcc and optimization flag -O
This flag does make the binary behave strangely and it is not recommended to use if if you compile the application on your own.
We have a patch provided by Richard Allen Wed Sep 28 1994 to compile libwww as a shared library.
