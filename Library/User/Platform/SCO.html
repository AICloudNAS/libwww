Notes About SCO

These are the installation and user notes for the CERN HTTP Server, the Library of Common Code and the Line Mode Browser. Any comments and suggestions are welcome at libwww@w3.org.

Compilation
Before compiling the CERN server version 3.0 you will need to apply two patches if you are using the native C compiler on you SCO platform. See more information on the Bugs Page.
