Working Plan W3C Reference Library

This is the current work plan for new features and changed interfaces for the W3C Referene Library.

Next major release is scheduled for DECEMBER 1st 1995. This release will go into public domain JANUARY 1st 1996.

Plan for Next release

  • Enhanced support for Power Mac and Windows
  • Embed stream taking an HTML object and strips everything outside (including) <BODY>
  • Descriptions in directory listings
  • Content-Length counter stream
  • HTTP/1.1 reference implementation
  • New NNTP module with support for POST and persistent connections
  • Persistent caching scheme
  • Documentation
  • Make small examples of the code

Long Term (when specified)

Prototypes of this will appear as we get the specifications ready.
  • HTTP-NG reference implementation
  • HTML level 3 parser
  • URN/URC support
  • Content Negotiation

Henrik Frystyk, libwww@w3.org, October 1995
