Working List for the Library

This is a list of what's going on in the W3C Reference Library. It also is a strong indication of what's going on in the Line Mode Browser.

Next release is scheduled for DECEMBER 1995

Plan for Next release

  • Discussion about better API
  • Support for Power Mac, OS/2 and other systems
  • Content Negotiation
  • HTTP/1.1 reference implementation
  • Persistent caching scheme
  • HTML level 2 parser and internationalization
  • Hooks for security
  • Support of SMTP, WhoIs++, and NNTP

Long Term (when specified)

  • HTTP-NG reference implementation
  • HTML level 3 parser
  • URN/URC support
  • Object oriented APIs

Henrik Frystyk, July 1995, libwww@w3.org
