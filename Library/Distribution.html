Downloading the W3C Reference Library (libwww)

W3C distributes source code, documentation, and example applications for the W3C Reference Library. Access to the distribution files are provided through our HTTP and FTP services. The easiest is simply to download the distribution file in one of the following formats described below.

Libwww is distributed under the W3C rules for software distribution. This means that all W3C members have full access to the release 1 month before it becomes publicly available. If you are a W3C member then please jump to the our member area for the latest version.

The source code is distributed as a single distribution package which you can get in the following formats:

  • gzipped tar file via HTTP or FTP
  • compressed tar file via HTTP or FTP
  • zip file via HTTP or FTP

The FTP site does only contain publicly available versions. If you are looking for the latest member version then please use our HTTP service. The FTP site also contains old versions and diffs between versions. Please read the README file for a more detailed description of the contents of the distribution package.

Direct Access via Anonymous FTP

Our public domain archives are also available through anonymous FTP. If you do not have a FTP client in your Web application then you can get to it by making a traditional FTP session as follows: 

	> ftp ftp.w3.org
	Enter user name: anonymous
	Password: me@myhost.edu
	ftp> cd /pub/libwww
	ftp> binary
	ftp> get w3c-libwww-x.x
	ftp> quit
	>

where x.x is the highest version number you can find. This file contains the complete libwww distribution, so this is all you need!
