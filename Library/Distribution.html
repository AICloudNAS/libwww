Getting the W3C Source Code Distribution

There are several ways of down loading the W3C source code distribution packages, but the two most important are:
  • Using your Web client
  • Using a FTP client
If you are using a Web client then you can get the source code from the following places:
  • W3C Reference Library
  • CERN HTTPD
  • Line Mode Browser
  • Next Browser/Editor
  • Directly to our source archive

The FTP server at w3.org is temporarily offline. All information which would be available via FTP is still available through HTTP. When FTP is available, this notice will be removed. Thank you for your patience.

If you are using a FTP client then you can FTP anonymously to ftp.w3.org by doing the following: 

	> ftp ftp.w3.org
	Enter user name: anonymous
	Password: me@myhost.edu
	ftp> cd /pub/www/src
	ftp> binary
	ftp> get WWWLibrary.tar.Z
	ftp> get WWWLineMode.tar.Z
	ftp> get WWWDaemon.tar.Z
	ftp> quit
	>
You are now ready to install the software!

Henrik Frystyk, May 1995
