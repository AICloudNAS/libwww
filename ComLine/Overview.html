W3C Command Line Tool

The W3C Command Line Tool is a cross platform command line tool for accessing the Web. It is developed in C and serves as an example of a client application using the W3C Reference Library and as a test-tool for quick access to the Web from scripts, cron jobs etc. It gives a variety of possibilities for data format conversion, filtering etc. It is primarily intended as a test-tool for quick access to the Web or used in scripts, cron jobs etc.

Getting the Source Code
The latest source code is available in the following formats:
  • compressed tar file
  • gzip'ed tar file
  • zip file

You can find old versions of the source code and diffs together with many other things in our distribution archives and you can browse through the latest source files on our server! A packet contains the following components:

  • Installation guide
  • Source code
  • Platform dependent make files
  • Hypertext documentation
  • Copyright statements and other legal mumbo jumbo

You can see how to unpack and compile the software in the README file, and you can find more information on how the W3C Reference Library is designed in the Library Architecture and how to use it in the User's Guide.
