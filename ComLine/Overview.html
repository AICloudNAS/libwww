W3C - The Libwww Command Line Tool

The w3c is an example application that comes with libwww - the W3C Sample Code Library. It is a command line tool with easy HTTP access and support for PUT, POST, DELETE, TRACE and any other HTTP/1.1 method. It gives a variety of possibilities for data format conversion, filtering etc. and can easily be used in scripts, cron jobs etc.
