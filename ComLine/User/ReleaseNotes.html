W3C Command Line Tool 4.0 Release Notes

  • This is the first release of the W3C Command Line Tool. It is called version 4.0 as it follows the W3C Reference Library closely. It is a simple "proof of concept" application that has inherited a lot of the command line functionality of the Line Mode Browser. The main purpose is to demonstrate how to use the Library and to give a "hands-on" experience.

Henrik Frystyk, libwww@w3.org, November 1995
