The Web Commander is a Win32 application for getting, saving, and deleting documents remotely using HTTP/1.1. It allows the user to explicitly control the metadata describing the document to save including the language, type, charset, etc.
  • Binary updated September 8, 1998
  • Now solves the lost update problem!

The Web Commander is part of the the libwww - the W3C Sample Code Library and is available in both source and binary versions.

Binary Installation Guide

As an exception, the Web Commander is provided in a binary version as well as in source. In order to install the binary version, complete the following steps:

  1. Download the latest precompiled binary version
  2. Unzip the distribution file into your preferred location. All files in the zip file are packaged under the "bin" folder, so you may want to extract it into C:\, for example. That way, the Web Commander will end up as "C:\bin\WinCom.exe".
  3. There is no installation wizard, but if you want to make a desktop shortcut then you can make a shortcut in the "bin" folder and drag it to the desktop.
  4. If ever you want to uninstall the Web Commander then just delete the "bin" folder (make sure that you don't have anything else there, of course)

Features

  • Uses the libwww for saving, getting, and deleting documents using HTTP/1.1
  • Explicit control over metadata describing the source that you are saving like language, charset, etc.
  • Support for version conflict detection - solves the lost update problem!
  • Support for proxies including ssh tunnelling
