WinCom

The W3C libwww Web Commander GUI Tool

WinCom is an example MS Windows 95/NT GUI application for doing PUT, DELETE and other HTTP/1.1 editing related operations. It is essentially the same idea as the Command Line Tool but the user has much better control over how to configure it and what metadata to use. It comes as a part of libwww - the W3C Sample Code Library.
